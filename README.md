# Miracle's Apartment Rentals

A modern, responsive web application for browsing and searching apartment rentals. Find your perfect home with an intuitive interface and beautiful design.

## Features

- **Responsive Design** - Fully mobile-friendly with adaptive layouts for all screen sizes
- **Search Availability** - Check in/check out date selection with occupancy filters (adults and kids)
- **Mobile Navigation** - Toggle sidebar menu for easy navigation on small screens
- **Social Media Integration** - Open Graph and Twitter Card meta tags for proper link sharing
- **Modern UI** - Clean, professional design with Font Awesome icons and Raleway typography
- **Apartment Showcase** - High-quality images of apartments including bedrooms, dining rooms, and living areas

## Getting Started

1. Open `index.html` in your web browser
2. Use the sidebar to search for available apartments by:
   - Selecting check-in and check-out dates
   - Specifying number of adults and kids
   - Clicking "Search availability"
3. Navigate using the top navbar menu on mobile devices

## Features in Detail
